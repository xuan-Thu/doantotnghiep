1. composer install
2. cp .env.example .env

4. Set up database in .env file
5. Import db file(database/doantotnghiep.sql) into your database
6. npm install
7. npm run watch
8. delete folder 'storage' in folder 'public'
9. php artisan storage:link
10. php artisan serve
